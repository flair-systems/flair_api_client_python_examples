# Flair API Client Examples (Python)

## Requirements
+ Python 3.5
+ virtualenv

## Usage

```bash
git clone git@github.com:flair-systems/flair_api_client_python_examples
cd flair_api_client_python_examples
virtualenv python=python3 .
source bin/activate
pip install -r requirements.txt
jupyter notebook
```
